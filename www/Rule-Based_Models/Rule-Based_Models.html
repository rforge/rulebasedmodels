This project hosts Cubist and C50, two packages based on the work of Ross Quinlan and are commercially released by RuleQuest software. These packages are based on the GPL versions of the software released in 2011.


Cubist is regression rule model based on the initial work on model trees described in Quinlan (1992) and Quinlan (1993a). Cubist decomposes the model tree into a set of rules and also has a boosting-like ensemble method call committees. The Cubist tutorial has more information, as does the package vignette.


C5.0 is an evolution of the C4.5 model describe in Quinlan (1993b). This model has several new features, such as winnowing for feature selection, boosting as well as using a cost function for building the model. C5.0 can create tree- and rule-based models. The C5.0 tutorial has details on how to use the model. The R package is almost complete.



References:


  1. Quinlan R (1992). “Learning with Continuous Classes.” Proceedings of the 5th Australian Joint Conference On Artificial Intelligence, pp. 343–348.


  3. Quinlan R (1993a). C4.5: Programs for Machine Learning. Morgan Kaufmann Publishers.


  4. Quinlan R (1993b). “Combining Instance–Based and Model–Based Learning.” Proceedings of the Tenth International Conference on Machine Learning, pp. 236–243.
